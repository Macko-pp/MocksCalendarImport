Crear Calendario para Mocks (archivo .ics)

Presiona el boton para copiar el prompt.

Pega el texto copiado en ChatGPT y escribe las materias que estas tomando en el Diploma.

Ejemplo:

┌─────────────────────────────────────────────────────────────────────────────┐ │ [...] │ │ DTEND;TZID=America/Bogota:20241209T140000 │ │ DESCRIPTION:Duration: 4 hours │ │ END:VEVENT │ │ │ │ END:VCALENDAR │ │ " │ │ End example. │ │ │ │ These are the classes I participate in, I don't take the exams of │ │ any other subjects: │ │ │ │ Physics HL │ │ Buisness HL │ │ Design Tech HL │ │ Spanish A SL │ │ Math Analysis SL │ │ English A SL │ └─────────────────────────────────────────────────────────────────────────────┘

Copia lo que ChatGPT te escribio y pegalo en el area de texto abajo, y presiona el boton de descargar.





El archivo que se descarga lo vas a abrir con tu app de calendario (si estas en celular, presiona el boton de compartir y selecciona tu app de calendario). Revisa que todo este bien y ya! Ahora tienes todos tus examenes Mock en tu calendario.
